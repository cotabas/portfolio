what were we Going to watch?

Ruby on Rails project for keeping track of all the movies you talk about watching then forget when you go to watch a movie. It also lets you see at a glance with movies are available on your streaming service. Written in Ruby and Javascript using The Movie Database api and the Pico.css framework. Currently running on a raspberry pi in my house, but feel free to make use of it if this is a problem you also have

Onlyfams.photos

This is an invite only image sharing site for my family to use. Written in Ruby, it takes advantage of Rails, Dropzone, ImageMagick, Devise, and Bootstrap. The images are stored in an amazon S3 bucket it was originally hosted on an AWS ec2 but now lives on a google cloud instance

Boppa PDX

Boppa PDX is a popup supper club in Portland, OR. I created this simple site for them with a little bit of functionality added using PHP and Javascript. This is a great example of how much vaule you can get from a simple site when on a tight budget.
